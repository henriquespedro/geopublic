This geographic based Public Participation platform – Have your say is available on the cloud under the scope of the European project Storm Clouds. This project aims to move public services to the cloud.

Cities like Valladolid and Thessaloniki, besides Águeda, are implementing services on the cloud.

Browser compatibility

This application uses standard HTML, CSS and Javascript. It runs fine on Firefox, Chrome ou Microsoft Edge. The Microsoft Internet Explorer is not supported.

You should have a display resolution of 1024x768 to have a better view of the underlying maps.
