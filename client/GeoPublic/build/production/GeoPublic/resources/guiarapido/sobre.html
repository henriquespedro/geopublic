A Plataforma de Participação Pública – Eu Participo encontra-se disponível através da cloud no âmbito do projeto internacional Storm Clouds, que objetiva levar os serviços públicos para a “nuvem”.

Cidades como Valladolid e Thessaloniki, para além de Águeda, estão a disponibilizar serviços públicos na “nuvem”, contribuindo para que outras cidades possam implementar estes mesmos serviços gratuitamente, sobre tecnologias de código aberto.

Compatibilidade com os browsers

Esta aplicação utiliza apenas HTML, CSS e Javascript standard. Corre em browsers que suportem os standards do consórcio W3C, como o Firefox, Chrome ou Microsoft Edge. O browser Microsoft Internet Explorer não é suportado.

Convém ter uma resolução superior ou igual a 1024x768.
