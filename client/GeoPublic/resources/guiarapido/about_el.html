Η γεωγραφική πλατφόρμα συμμετοχής στα κοινά Πες τη γνώμη σου παρέχεται στο πλαίσιο του Ευρωπαικού Προγράμματος Storm Clouds. Το πρόγραμμα έχει ως σκοπό τη μεταφορά δημοσίων υπηρεσιών στο cloud.

Εκτός από τη Θεσσαλονίκη, οι πόλεις Valladolid, Agueda και Miscolc υλοποιούν υπηρεσίες στο cloud.

Συμβατότητα με φυλλομετρητές

Η εφαρμογή χρησιμοποιεί HTML, CSS και Javascript. Εκτελείται καλά στους φυλλομετρητές Firefox, Chrome και Microsoft Edge. Ο Microsoft Internet Explorer δεν υποστηρίζεται.

Θα πρέπει να έχετε ρυθμίσει την ανάλυση σας τουλάχιστον στα 1024x768 για να έχετε μία καλή αποτύπωση των χαρτών.
